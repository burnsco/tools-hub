# Tools Hub

Tools Hub is a browser-based collection of developer utilities built with Astro. The focus is quick access, local-first execution, and a more polished UI than typical utility sites.

## Highlights

- Generators, converters, and small workflow helpers
- Privacy-first browser execution
- Static-first Astro architecture with fast load times

## Stack

- Astro
- Tailwind CSS
- Bun

## Quick start

```bash
bun install
bun run dev
```

Build and deploy:

```bash
bun run check
bun run build
```
